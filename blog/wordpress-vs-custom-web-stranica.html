WordPress vs. Custom Web Stranica — Što je Bolje?

ENKR tim · 24. veljače 2025. · 6 min čitanja

Kada poduzetnik u Hrvatskoj odluči napraviti web stranicu, jedno od prvih pitanja koje si postavi je: "Trebam li WordPress ili nešto custom?" WordPress pokreće oko 43% svih web stranica na internetu, što ga čini daleko najpopularnijim CMS-om. No popularnost ne znači uvijek da je to najbolji izbor za vaš posao.

U ovom članku detaljno uspoređujemo WordPress s custom web stranicama izrađenim pomoću modernih tehnologija poput Astro i Next.js — frameworka koje koristimo u ENKR-u. Cilj je pomoći vam donijeti informiranu odluku.

Brzina — Zašto je važna?

Google je još 2021. uveo Core Web Vitals kao faktor rangiranja. Stranica koja se učitava duže od 3 sekunde gubi 53% mobilnih posjetitelja. WordPress stranice, posebno one s mnogo pluginova i velikom temom, rijetko postižu Lighthouse score iznad 70.

Custom stranice izrađene u Astro-u generiraju statički HTML s minimalnim JavaScriptom. Rezultat? Lighthouse score redovito iznad 95/100, učitavanje ispod jedne sekunde i značajno bolje pozicije na Googleu.

Sigurnost — Skriveni rizik WordPressa

WordPress je najčešće hakirana platforma na svijetu. Razlog je jednostavan: tisućama pluginova upravlja zajednica volontera, a mnogi se ne ažuriraju redovito. Prema izvještaju Sucuri-ja, 96.2% svih zaraženih CMS stranica bile su WordPress stranice.

Custom stranica nema bazu podataka, nema admin panel, nema pluginove — nema što hakirati. Statički HTML serviran preko CDN-a praktički je neprobojan.

Troškovi — Cijela slika

Mnogi poduzetnici gledaju samo inicijalnu cijenu, ali pravi trošak web stranice vidljiv je tek kroz godinu ili dvije:

  • WordPress: hosting (15–50€/mj), premium tema (50–100€/god), premium pluginovi (50–200€/god), SSL certifikat, sigurnosni backup servis, redovito održavanje
  • Custom stranica: domena (10–15€/god), hosting na Vercelu ili Netlifyu (0€ za male stranice), bez licence za teme ili pluginove

Nakon 2 godine, WordPress vas može koštati 600–1.600€ samo za hosting i licence, dok custom stranica ostaje na 20–30€ (cijena domene).

SEO — Tko pobjeđuje na Googleu?

WordPress ima Yoast SEO i slične pluginove koji pomažu s osnovnim SEO postavkama. Međutim, ti pluginovi ne mogu kompenzirati spore stranice i loše Core Web Vitals rezultate.

Custom stranice nude SEO prednosti iz temelja: semantički HTML, optimizirane slike, strukturirani podaci (schema.org), brzo učitavanje i čiste URL strukture — sve bez potrebe za dodatnim pluginovima. Google algoritam jasno favorizira brže i tehnički čišće stranice.

Održavanje — Koliko vremena trošite?

WordPress zahtijeva redovito ažuriranje jezgre, teme i svakog pojedinog plugina. Jedan nekompatibilni update može srušiti cijelu stranicu. Prema istraživanju WP Engine-a, 73% WordPress stranica ima barem jedan zastarjeli plugin s poznatim sigurnosnim propustima.

Custom stranica ne zahtijeva ažuriranja. Sadržaj se može mijenjati kroz jednostavan Git commit ili headless CMS, bez rizika od rušenja stranice.

Pluginovi — Blagoslov ili prokletstvo?

WordPress ekosustav nudi preko 60.000 pluginova, što zvuči impresivno. U praksi, svaki plugin dodaje JavaScript, CSS i bazu upita koji usporavaju stranicu. Prosječna WordPress stranica koristi 20–30 pluginova, od kojih mnogi imaju preklapajuće funkcionalnosti.

S custom pristupom, svaka funkcionalnost se implementira ciljano — bez viška koda, bez konflikata između pluginova i bez mjesečnih pretplata za premium verzije.

Kada je WordPress ipak bolji izbor?

WordPress ima svoje mjesto, i bilo bi nepošteno to ne priznati. Ako trebate:

  • Velik blog s više autora — WordPress admin sučelje je intuitivno za timove koji svakodnevno objavljuju sadržaj
  • Složen e-commerce — WooCommerce nudi tisuće integracija za velike web trgovine
  • Česte samostalne izmjene sadržaja — ako nemate developera i želite sami uređivati stranicu svaki dan

Za veće content-heavy stranice s desecima novih objava tjedno, WordPress je i dalje solidan izbor — pod uvjetom da investirate u kvalitetan hosting i redovito održavanje.

Zaključak

Za većinu malih poduzetnika u Hrvatskoj, custom web stranica je pametnije ulaganje. Brža je, sigurnija, jeftinija za održavanje i bolje rangirana na Googleu. WordPress ostaje relevantan za velike sadržajne portale i složene e-commerce projekte, ali za prezentacijske stranice, landing pageove i manje poslovne stranice — custom pristup pobjeđuje na svim frontovima.

Ključno je ne gledati samo inicijalnu cijenu, već ukupni trošak vlasništva kroz 2–3 godine. Kada uzmete u obzir hosting, licence, održavanje i izgubljene posjetitelje zbog spore stranice — custom rješenje se isplati od prvog dana.
